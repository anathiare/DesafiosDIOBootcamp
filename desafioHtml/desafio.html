Conteúdo 1 - Introdução ao HTML

Voltar | Próximo | Voltar ao início

O HTML é uma linguagem de marcação utilizada para criar páginas web. Ela é composta por tags, que são elementos que definem o conteúdo da página.

Veja mais...

A tag <html> é a tag principal que define o início e o fim do documento HTML.
A tag <head> é a tag que define o cabeçalho do documento HTML.
A tag <body> é a tag que define o corpo do documento HTML.
A tag <div> que é uma tag genérica para uso geral, mas que deve ser utilizada com moderação por conta da semântica.

Conteúdo 8 - Trabalhando com Áudios em HTML

Voltar | Próximo | Voltar ao início

Apredemos a utilizar os áudios para melhorar a apresentação do conteúdo, podendo ter um controle de volume, tempo de reprodução, entre outros. Além de poder deixar rolando o áudio enquanto o usuário navega pela página.

Veja mais...

A tag <audio> é a tag que define um áudio do documento HTML.
A tag <source> é a tag que define um áudio do documento HTML.
Adicionar controls para adicionar os controles de volume, tempo de reprodução, entre outros.
Adicionar autoplay para reproduzir o áudio automaticamente.
Adicionar loop para repetir o áudio.
Adicionar muted para silenciar o áudio.
Adicionar preload para carregar o áudio antes de reproduzi-lo.
Adicionar controlslist para adicionar os controles de volume, tempo de reprodução, entre outros.
